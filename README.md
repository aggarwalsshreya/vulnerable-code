# vulnerable-code
This repository contains a deliberately vulnerable Python web application built with a Flask-style structure. It is designed to test static analysis tools by including a wide range of security vulnerabilities distributed across multiple layers of the application.
The codebase mimics a real-world backend with routes, services, database access, utilities, and background jobs. Vulnerabilities are not isolated to single files—they often span multiple modules to require inter-file and data-flow analysis.
